# Sales Performance Dashboard

## Objective
To analyze real-world sales data and build a dashboard that helps track revenue,
sales trends, regional performance, and product-level insights for business decisions.

## Dataset
The dataset contains order-level sales data with the following fields:
- Order ID
- Order Date
- Product
- Category
- Quantity
- Price
- Region

Revenue is calculated using quantity Ã— price.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- KPI summary including total revenue, total orders, and average order value
- Monthly revenue trend analysis
- Region-wise sales performance analysis
- Product-level revenue analysis
- Category-wise revenue contribution

## Key Business Insights
- Revenue trends highlight overall business growth over time
- Certain regions outperform others in terms of sales
- A small number of products contribute a significant portion of total revenue
- Category-level analysis helps guide inventory and marketing decisions

## Outcome
This project demonstrates practical data analysis skills and the ability to
translate raw sales data into actionable business insights.
